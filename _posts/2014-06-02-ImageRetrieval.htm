A3: Leveraging Semantics with Attribute-associated Auxiliary References for General Image Retrieval

Being submission

This paper develops a general framework of image retrieval, named A3, by introducing an auxiliary set of samples (image references), each of which is annotated with semantic attributes (tags). Given a query image (without tags), we first map it into the references by a non-convex sparse coding formulation, which jointly optimizes appearance reconstruction of the query and semantics consistency among references. A novel algorithm, namely semantic coherent coding (SCC), is presented to solve this optimization. As a result, a subset of references is selected to augment the query for retrieval. Then we can rank images to be retrieved by intuitively matching them together with the query and the constructed subset. The advantages of our approach are summarized as follows. (1) The auxiliary set of references gives rise to an alternating way for query augmentation while utilizing semantics. (2) The semantics of the references can be also propagated into the query and database images.

Experiments

Datasets: We validate the framework A3 on two public databases: Caltech-256 [20] and ImageNet [21]. The former dataset is employed for evaluating the performances of image retrieval, as most existing approaches also reporting results on it. We demonstrate another advantage of A3, i.e. semantic attribute discovery, on the ImageNet database, in which many images are not yet annotated with attributes. For both datasets, we split the data into auxiliary and retrieval set, use the auxiliary set(with annotated attributes) to augment the query through semantic-aware reconstruction and use the retrieval set to evaluate the performance. Semantic Attributes: We collect totally 81 attributes grouped into 5 categories ( including color, pattern, shape, part and material ), which represent image semantic from different aspects.
